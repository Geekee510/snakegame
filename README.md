# Snake Game

Classic Snake game built with Vite + React + MUI + Tailwind CSS.

## Features
- Arrow key controls
- Score tracking with localStorage high score
- Speed increase as you eat more food
- Pause/Resume with Space key
- Touch support for mobile
- Dark theme with grid background

## Play Online
[Play on GitHub Pages](https://geekee510.github.io/snakegame/)

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
